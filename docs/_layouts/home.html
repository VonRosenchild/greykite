api

Easy to start

Greykite has a straightforward API and several default templates and parameters. Additionally, Greykite comes with tutorials and quick-start examples that help you obtain forecasts with only a few lines of code.

flash_on

Fast algorithm

The core algorithm, named Silverkite, uses a regression-based algorithm to fit the time series model. You can obtain the solution as fast as performing a regression.

construction

Flexible design

The algorithm comes with many popular time features, such as seasonality fourier series and holiday indicators, to be used as regressors. Moreover, you can freely create any features that are related to the time series, and use them to fit the machine learning model of your choice.

double_arrow

Full functionality

We provide tools from exploratory data analysis (EDA) to model pipeline, which include powerful plots, data transformation, model selection, and benchmarking. Moreover, we have automatic changepoint detection and model summary that can be used independently from the forecasting method.
